# SEO GSC Keyword Clusters

## WhatsApp
- Primary keywords: WhatsApp marketing software, WhatsApp sender software, WhatsApp Business API platform, WhatsApp Business API included, WhatsApp automation.
- Target landing pages: `/whatsapp-marketing-software`, `/whatsapp-sender-software`, `/whatsapp-business-api-included`, `/crm-with-whatsapp-automation`.
- Optimization actions:
  - High impressions + low CTR: tighten titles to include “API included” and “opt-in”, add FAQ rich snippets on WhatsApp pages, test benefit-first meta descriptions.
  - Rankings 8–20: add internal links from `/all-in-one-marketing-automation-platform`, `/pricing`, and `/automation-funnels` with anchor text “WhatsApp automation”.
  - Internal linking rules: every funnel, CRM, and workflow page should link to `/whatsapp-marketing-software`.

## Funnels
- Primary keywords: AI funnel builder, automation funnels, webinar funnels, VSL funnels, lead funnels.
- Target landing pages: `/ai-funnel-builder`, `/automation-funnels`, `/ai-landing-page-builder`.
- Optimization actions:
  - High impressions + low CTR: add outcome-driven snippets (“launch in minutes”, “AI auto-funnel generation”) and include funnel types in H2s.
  - Rankings 8–20: add links from `/payment-gateway-integrations`, `/crm-with-whatsapp-automation`, `/all-in-one-marketing-automation-platform`.
  - Internal linking rules: every funnel page should link to `/payment-gateway-integrations` and `/crm-with-whatsapp-automation`.
